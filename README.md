# SiriLikeAssistant

Full Android Studio project ready to upload to GitHub.

## How to use
1. Upload this repository to GitHub (upload the ZIP contents to a new repo or push using git).
2. Ensure the default branch is `main` (or change the workflow to `master`).
3. GitHub Actions will run the workflow and build the APK. Download the artifact from the Actions run.

## Build locally
Open in Android Studio and build APK.
